# Gankki
干货集中营第三方安卓客户端，练练手~
## 简介
+ 数据来自[干货集中营](http://gank.io)公开API
+ 浏览、随机浏览、搜索、收藏
+ Material Designer设计
+ MVP架构，Retrofit2+RxJava2
